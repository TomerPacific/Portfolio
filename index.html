What I Do

Mobile Development

I have been working in a mobile setting for the past six years. I am currently working in developing a SDK (Software Development Kit) for both platforms (Android/iOS) that gets used by millions of users. I have also released several applications to the Google Play Store, written in Java/Kotlin/Dart. Ranging from fun applications, like enabling users to contact Israel's Minister of Transportation, to allowing users to generate XML code for Android menus. I have also been working with Flutter publishing several applications and a plugin for it that makes it easier to use a dropdown widget.

Frontend Development

As part of my current job, I am also working in a Frontend capacity, using Typescript. In my spare time, I try to learn the fundamentals of the newer frameworks (React/Vue/Angular) by creating various projects. You can see some of these projects in my CodePen account or in my GitHub profile.

Backend Development

As part of the projects that I work on in my free time, I am also working with NodeJS. I have created several backend applications, that are deployed using Heroku, that serve projects that I have created. One example is of a service that checks the status of all of my projects to see if they are online.
